# 更新日誌

## 2026-06-02

### 新增
- 新增 GIF 小片段製作功能。
- 支援以頁面快捷鍵開啟 GIF 製作介面。
- 支援設定 GIF 解析度與 FPS。
- 支援依目前播放時間快速帶入 GIF 開始時間。
- 支援以開始時間、結束時間、秒數來建立 GIF 片段。

### 調整
- GIF 製作介面已移到較直覺的主畫面與頁面泡泡流程。
- GIF 輸出流程已改為直接從影片影格擷取，不包含聲音。
- 修正部分 GIF 檔案在嚴格解碼器下無法開啟的問題。

### 已知問題
- Edge 目前仍可能無法正確讀取動畫瘋播放器時間軸，這個問題尚未完全修復。
- GIF 畫質仍有改善空間，後續需要再優化影格取樣與色彩壓縮品質。
